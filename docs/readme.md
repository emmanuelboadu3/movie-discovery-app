## Overview
A centralized movie and TV discovery application that helps users:
- Browse trending content
- Search by title, genre, or year
- View detailed info (synopsis, cast, trailers, streaming availability)
- Manage watchlist and favorites
- Get personalized recommendations

## Features
- TMDb API integration
- Streaming availability via Utelly/JustWatch
- LocalStorage persistence
- Responsive design with dark theme + neon accents
- Error handling and loading states
- Unit tests for watchlist and favorites

## Installation
1. Clone the repository
2. Run locally with a simple server (e.g., `npx serve`)
3. Open `index.html` in your browser

## Deployment
- GitHub Pages or AWS Elastic Beanstalk
